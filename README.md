## Comandos
* Caso não tenha instalado o react native
    * npm i -g react-native
* Criação do projeto
    * npx expo init app
* Entrar na pasta
    * cd app
* Iniciar projeto
    * expo start
* Navigation
    * https://reactnavigation.org/docs/getting-started/
    * expo install react-native-screens react-native-safe-area-context
